mapjumper by dudarev

Chrome extension to jump from one map service to another.

Install directly from Chrome Web Store

How to use

Install dev version

In Chrome:
Tools > Extensions > Developer mode > Load unpacked extension > Browse to the src directory.

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/dudarev/mapjumper
