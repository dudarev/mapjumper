mapjumper by dudarev

Chrome extension to jump from one map service to another.

Chrome extension to jump from one map to another. As of version 0.1 only jumping FROM: Gowalla spots TO: Wikimapia and Openstreetmap is implemented. More services and directions are planned.

Install

Download or clone the code with git. In Chrome: Tools > Extensions > Developer mode > Load unpacked extension > Browse to the src directory.
