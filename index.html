Mapjumper

Chrome extension to jump from one map service to another

Install directly from Chrome Web Store or use instructions below to install from source code on Github.

How to use

Download

Clone the project by running: 

$ git clone git://github.com/dudarev/mapjumper

Use dev branch for the latest features: 

$ git checkout dev

Install

In Chrome navigate to 

      Tools >
      Extensions > 
      Developer mode > 
      Load unpacked extension
Select mapjumper/src directory.
